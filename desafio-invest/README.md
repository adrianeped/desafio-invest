# :tw-1f4b8: Carteira de Investimentos

### Sobre:

- Essa é uma aplicação que simula uma carteira de investimentos onde é possível comprar, vender, visualizar os ativos e o resumo total investido.

### Como usar:

```bash
dotnet build
dotnet run
```

### Tecnologias

- Linguagem: C#
- Framework: .NET Core
